# Angular Weather App

This project was coded with Angular version 11.2.2.


## System requirements for build
* nodejs
* npm | yarn
* angular

## Get Started

### To get started, run next command:

`npm install`


### To start Development server, run:

`npm start` of `ng serve` and navigate to `http://localhost:4200/`


## Build

### To build the project, run:

$ ng build

The build artifacts will be stored in the `dist/` directory. Use the `--prod` flag for a production build.


## Running unit tests

### To run unit tests:

$ ng test


## Tp run end-to-end tests:

$ ng e2e
